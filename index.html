CSS Selectors

  • p Element selector has no prefix
  • # #id Use # to select elements identified by id. <p id="id">
  • . .class Use dot to select elements identified by class. <p class="class">
  • [] img[src] Use Element[attribute] to select elements with that attribute.
  • : a:hover Use Element:State to select the element when it's in that state.
